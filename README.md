# 🌟 Pattern Generator and Number Analyzer 🌟

## 📖 Overview

### This is a simple and interactive **Python menu-driven program** that allows users to:

#### ✅ Generate star patterns
#### ✅ Analyze numbers in a given range
#### ✅ Identify Even and Odd numbers
#### ✅ Calculate the total sum of numbers

### The project is beginner-friendly and demonstrates the use of important Python concepts such as:

#### • 🔁 Loops
#### • 🔀 Conditional Statements
#### • 📥 User Input
#### • 🖨️ Output Formatting

## ✨ Features

### ⭐ 1. Generate a Pattern

#### • Accepts the number of rows from the user
#### • Prints a right-angled star pattern

### 🔢 2. Analyze a Range of Numbers

#### • Takes starting and ending numbers from the user
#### • Checks whether each number is Even or Odd
#### • Calculates the total sum of all numbers in the range

### ❌ 3. Exit

#### • Safely terminates the program
